## 🔥 Projetos em destaque

### 🟦 Lazy - Utilitário de Comando e Monitor Serial
O que esse projeto demonstra:
- Desenvolvimento de ferramentas **CLI** para otimização de fluxo de trabalho no terminal.
- Lógica e configuração de **baud rate serial** para comunicação com hardwares.
- Gerenciamento eficiente de pacotes e bibliotecas nativas.

**Stack:** Go 

---

### 🟪 Termostato PID (Automação Cervejeira)
O que esse projeto demonstra:
- **Automação IoT** utilizando Arduino/ESP32 com sensores de temperatura (ex: DS18B20).
- Implementação de lógica de controle **PID** para estabilidade térmica precisa durante a brassagem.
- Integração prática entre código de baixo nível e hardware.

**Stack:** C++ • Arduino
